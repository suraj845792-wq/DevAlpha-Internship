## 🗂️ Tasks Overview

### ✅ Task 1 (Easy) — OOP Based Program
**Project: Bank Account Management System**

A full-featured banking system built using core OOP principles.

**Concepts Used:**
- Classes & Objects
- Encapsulation
- Inheritance (SavingsAccount, CurrentAccount)
- Polymorphism (virtual functions)
- Abstraction (abstract base class)
- Exception Handling

**Features:**
- Create Savings & Current accounts
- Deposit / Withdraw with validations
- Minimum balance enforcement
- Overdraft support
- Monthly interest calculation
- Transaction history with timestamps

---

### ✅ Task 2 (Easy) — Data Structure Application
**Project: Student Grade Management System**

A student records system using arrays and multiple sorting/searching algorithms.

**Concepts Used:**
- Arrays & Structures
- Bubble Sort (by percentage)
- Selection Sort (by name)
- Linear Search (by name)
- Binary Search (by roll number)

**Features:**
- Add / Delete students
- Enter marks for 5 subjects
- Auto grade calculation (A+ to F)
- Sort by percentage or name
- Search by name or roll number
- Class statistics (average, highest, lowest)

---

### ✅ Task 3 (Intermediate) — Management System Project
**Project: Hotel Management System**

A complete hotel management solution built using OOP and STL.

**Concepts Used:**
- Classes & Objects (Room, Guest, Booking, Hotel)
- Vectors (dynamic storage)
- Exception Handling
- Encapsulation
- Date & Time handling

**Features:**
- 11 rooms (Single / Double / Suite)
- Guest registration with unique ID
- Room booking with bill calculation
- Checkout with invoice generation
- Hotel summary (revenue, occupancy)
- View all bookings and guests

---

### ✅ Task 4 (Advanced) — Advanced C++ Application
**Project: Mini Compiler / Expression Calculator**

A real-world expression parser built like an actual compiler with Lexer and Parser stages.

**Concepts Used:**
- Lexer / Tokenizer (compiler design)
- Recursive Descent Parser
- Operator Precedence handling
- STL (Stack, Map, Vector)
- Templates & Enums
- Exception Handling
- OOP (Lexer, Parser, Calculator classes)

**Features:**
- All arithmetic operators: `+ - * / % ^`
- Math functions: `sin cos tan sqrt log ln abs`
- Constants: `pi`, `e`
- Variable assignment: `x = 10`
- Last answer: `ans`
- Calculation history
- Error messages for invalid input

## ⚙️ How to Run

### Prerequisites
- G++ Compiler (MinGW for Windows)
- VS Code or any C++ IDE

### Compile & Run
```bash
# Task 1
cd Task1_OOP_BankSystem
g++ bank_management.cpp -o bank_management
.\bank_management

# Task 2
cd Task2_DSA_GradeSystem
g++ student_grade.cpp -o student_grade
.\student_grade

# Task 3
cd Task3_Hotel_Management
g++ hotel_management.cpp -o hotel_management
.\hotel_management

# Task 4
cd Task4_Advanced_MiniCompiler
g++ mini_compiler.cpp -o mini_compiler -lm
.\mini_compiler
```
